# **App Name**: InfoKegiatanSemarang

## Core Features:

- Activity Logging: Users can log activities with details such as description, duration, and time of day.
- Schedule Generation: AI generates an optimized daily or weekly schedule based on the user's logged activities and priorities, serving as a tool to manage activities better.
- Calendar View: Display activities in a calendar view for easy visualization.
- Priority Setting: Allow users to set priorities for different activities, which will influence the generated schedule.

## Style Guidelines:

- Primary color: Red (#FF0000) for a sense of energy.
- Background color: Yellow (#FFFF00).
- Accent color: Lavender (#B284BE) to complement the primary, providing subtle contrast and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif combining a modern look and some warmth; it works well for both headlines and body text.
- Use simple, clear icons to represent different types of activities.
- Clean and intuitive layout for easy navigation and focus on scheduling.
- Subtle transitions and animations to enhance user experience when navigating the app.
